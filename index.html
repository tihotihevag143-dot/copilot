    Paste that in a Windows PowerShell prompt.

  • How to Install


    1
    Open the Terminal app
    Press ⌘ Space to open Spotlight, type Terminal, and press Enter.
    2
    Copy the command
    Click the 📋 button above. You can also click the command itself to select it, then press ⌘ C.
    3
    Paste it into Terminal and run
    Switch back to Terminal, press ⌘ V to paste, then press Enter.
    4
    Follow the prompts
    If asked, type your Mac login password (characters won’t show as you type) and press Enter. Wait a few seconds for the installer to finish — that’s it.

    On Linux, use Ctrl instead of ⌘.

  • How to Install


    1
    Open PowerShell
    Press Win + R, type powershell, and press Enter.
    2
    Copy the command
    Click the 📋 button above. You can also click the command itself to select it, then press CtrlC.
    3
    Paste it into PowerShell and run
    Switch back to PowerShell, press CtrlV to paste, then press Enter.
    4
    Follow the prompts
    If asked for admin permission, click Yes. Wait a few seconds for the installer to finish.
